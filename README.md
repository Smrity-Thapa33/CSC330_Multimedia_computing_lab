# CSC330: Multimedia Computing Lab Works

This repository contains implementations and experiments performed as part of the **CSC330 Multimedia Computing Lab** course. It covers fundamental concepts of multimedia systems including image processing, audio processing, video representation, animation, and compression techniques.

---

##  Objectives

- To understand digital representation of multimedia data (image, audio, video)
- To implement basic image processing and enhancement techniques
- To explore animation and motion concepts
- To learn compression techniques used in multimedia systems

---

##  List of Experiments

###  Image Processing
- Digital Image Representation and Processing  
- Image Histogram Analysis (Pixel Intensity Distribution)  
- 2×2 Cluster Dot Halftoning for Digital Image Dithering  
- Image Display Using Abstraction in OpenCV  
- Webcam Image Capture using OpenCV  

---

###  Audio Processing
- Sound Digitization  
  - Recording audio  
  - Displaying waveform  
  - Sampling and quantization  

---

###  Video Processing
- Digital Representation of Video Signals  
  - Resolution  
  - Frame Rate  
  - Frame Count  

---

###  Animation Techniques
- Animation of Circular Motion using Sine and Cosine Functions  
- Linear Motion of Objects using Tweening (In-betweening) Techniques  

---

###  Compression Techniques
- Run Length Encoding (RLE)  
- Huffman Encoding  
- JPEG Compression & Decompression using OpenCV  

---

##  Tools & Technologies Used

- Python
- OpenCV
- NumPy / Matplotlib 
- Audio processing libraries
